# Black Notes
A ReactNative note app

![app example](http://i.imgur.com/zj361Dml.png)

## SETUP

```
$ git clone https://github.com/bdougie/BlackNotes
$ npm install
```

### Firebase

- Create a firebase app at Firebase.com
- Place firebase url on line 4 of App/Lib/Api.js

## Xcode

- Run simulator
